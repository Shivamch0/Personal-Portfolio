🚀 Personal Portfolio Website

A modern and interactive 3D Developer Portfolio built with React, TypeScript, and Three.js to showcase projects, skills, experience, and achievements with immersive visuals, smooth animations, and a fully responsive design.

Designed to create a strong developer presence while demonstrating modern frontend development practices and interactive UI experiences.


✨ Features
🌌 Interactive 3D visuals powered by Three.js
⚡ Lightning-fast performance with Vite
🎨 Modern and clean responsive UI
✨ Smooth animations and page transitions
📱 Fully mobile-friendly experience
🧩 Reusable and scalable React components
📂 Dynamic project and experience showcase
🔗 Social media and GitHub integration
🛠️ TypeScript support for maintainable code
🚀 Optimized project structure for scalability


🛠️ Tech Stack

│

│

⚙️ Getting Started

1️⃣ Clone the Repository
git clone https://github.com/your-username/portfolio.git

2️⃣ Navigate to the Project Directory
cd portfolio

3️⃣ Install Dependencies
npm install

4️⃣ Start Development Server
npm run dev

The application will run locally at:

http://localhost:5173
🏗️ Production Build

Create an optimized production build:

npm run build

Preview the production build locally:

npm run preview
