🚀 Personal Portfolio Website
A modern and interactive 3D developer portfolio built using React, TypeScript, and Three.js.
This portfolio showcases projects, skills, experience, and contact information with smooth animations and responsive design.
✨ Features
🌌 Interactive 3D visuals using Three.js
⚡ Fast and optimized with Vite
🎨 Modern responsive UI
✨ Smooth animations and transitions
📱 Fully mobile responsive
🧩 Reusable React components
📂 Dynamic projects and experience sections
🔗 Social and GitHub links integration
🛠️ TypeScript support for better scalability
🛠️ Tech Stack
Frontend
React.js
TypeScript
Vite
Tailwind CSS
3D & Animation
Three.js
React Three Fiber
React Three Drei
Framer Motion

⚙️ Installation & Setup
1️⃣ Clone the Repository
Bash
git clone https://github.com/your-username/portfolio.git
2️⃣ Navigate to Project Directory
Bash
cd portfolio
3️⃣ Install Dependencies
Bash
npm install
4️⃣ Run Development Server
Bash
npm run dev
The application will start on:
Bash
http://localhost:5173
🏗️ Build for Production
Bash
npm run build
Preview production build:
Bash
npm run preview
